## 1. 系统概述

科研成果管理系统用于支撑科研成果的录入、归集、审核、检索、权限申请、成果类型配置、过程成果查看以及需求洞察、研究洞察等业务场景。

系统当前主要面向三类角色：

| 角色 | 系统角色标识 | 主要职责 |
| :--- | :--- | :--- |
| 普通用户 | `project_leader` | 创建成果物、提交审核、查看个人成果、检索成果、申请查看受限成果全文 |
| 专家用户 | `research_expert` | 查看待审核成果、下载附件、填写审核意见、执行审核通过或驳回 |
| 管理员 | `research_admin` | 查看科研看台、管理成果、分配审核人、处理权限申请、配置成果类型和动态字段、查看过程成果和研究洞察 |

## 3. 登录与退出

### 3.1 登录系统

1. 在浏览器中打开科研成果管理系统地址。
2. 如当前未登录，系统自动跳转到统一登录门户。
3. 在统一登录门户完成身份认证。
4. 系统验证登录票据后自动返回科研成果管理系统。
5. 系统根据用户角色进入默认页面：
   - 管理员默认进入`科研看台`。
   - 专家默认进入`成果审核`。
   - 普通用户默认进入`个人概览`。

![图 3-1 统一门户中选择科研成果管理系统](user-manual-assets/screenshots/00-login-portal.png)

![图 3-2 统一身份认证登录页](user-manual-assets/screenshots/01-login-account.png)

### 3.2 退出登录

1. 点击页面右上角用户信息区域。
2. 在下拉菜单中点击`退出登录`。
3. 系统弹出确认框，点击`确定`。
4. 系统清除本地登录信息并跳转回登录门户。

### 3.3 无权限提示

如果用户直接访问无权限页面，系统会跳转到`403`页面。出现该情况时，请确认账号是否具备对应角色权限。

## 4. 普通用户操作说明

### 4.1 查看个人概览

操作路径：`个人概览`

普通用户进入系统后，可在个人概览页面查看：

- 总成果数
- 论文数
- 专利数
- 本月新增成果数
- 成果类型分布图
- 近年成果产出趋势图
- 最近提交的成果列表

可点击页面中的`新建成果`进入成果创建流程，也可点击`查看全部`进入个人成果物列表。

![图 4-1 普通用户个人概览](user-manual-assets/screenshots/02-user-dashboard.png)

### 4.2 创建成果物

操作路径：`创建成果物`

成果创建采用分步式流程，当前包含五个步骤：`选择类型`、`填写基础信息`、`上传附件`、`智能补全`、`提交`。

#### 4.2.1 选择成果类型

1. 进入`创建成果物`页面。
2. 在`选择成果类型`下拉框中选择成果类型。
3. 系统会展示该类型的说明信息。
4. 点击`下一步`。

注意事项：

- 只展示已启用的成果类型。
- 如果管理员未配置某类成果的动态字段，系统会提示该类型暂未配置动态字段。

![图 4-2 创建成果物：选择成果类型](user-manual-assets/screenshots/03-create-step1-type.png)

#### 4.2.2 填写基础信息

1. 填写`成果标题`。
2. 填写或选择`作者`，支持多作者。
3. 选择`年份`。
4. 填写`摘要`。
5. 填写`关键词`，可输入后按回车添加多个关键词。
6. 如当前成果类型配置了动态字段，继续填写页面展示的扩展字段。
7. 点击`下一步`。

必填校验：

- `成果标题`为必填项。
- `作者`为必填项。
- `年份`为必填项。
- 管理员配置为必填的动态字段需要填写后才能进入下一步。

![图 4-3 创建成果物：填写基础信息](user-manual-assets/screenshots/04-create-step2-info.png)

#### 4.2.3 上传附件与设置可见范围

1. 在`附件上传`区域点击上传，或将文件拖拽到上传区域。
2. 系统支持一次选择多个附件。
3. 设置成果`可见范围`：
   - `私有`：默认范围，未授权用户只能查看有限信息或需要申请访问。
   - `公开`：具备访问权限的用户可查看公开内容。
4. 点击`下一步`。

附件限制：

- 当前前端限制单个附件大小不能超过 20MB。
- 如附件超过限制，系统会提示`单个附件不能超过 20MB`并移除该文件。
- 过程管理系统同步文件的服务端配置允许 `pdf`、`doc`、`docx`、`xls`、`xlsx`、`txt`、`zip`、`rar` 等类型，普通成果附件以页面实际上传结果为准。

![图 4-4 创建成果物：上传附件与设置可见范围](user-manual-assets/screenshots/05-create-step3-upload.png)

#### 4.2.4 智能补全

当前版本页面保留`智能补全`步骤，但功能状态显示为`暂未开发`。用户无需在该步骤填写内容，直接点击`下一步`进入确认提交。

交付说明：

- 当前版本不应将智能补全作为正式可操作功能验收。
- 如后续版本开放，手册应单独补充附件识别、字段预填和人工确认流程。

![图 4-5 创建成果物：智能补全暂未开放](user-manual-assets/screenshots/06-create-step4-autofill.png)

#### 4.2.5 确认并提交审核

1. 在`确认信息`页面核对成果标题、成果类型、作者、年份、摘要、关键词、可见范围、扩展信息和附件列表。
2. 确认无误后点击`提交审核`。
3. 系统提示`提交成功`后，自动跳转到`个人成果物`页面。
4. 新提交成果状态通常为`待审核`。

![图 4-6 创建成果物：确认信息并提交审核](user-manual-assets/screenshots/07-create-step5-confirm.png)

### 4.3 查看个人成果物

操作路径：`个人成果物`

页面支持按以下条件筛选：

- 状态：全部、草稿、待审核、审核中、已驳回、已发布
- 关键词

列表展示内容包括：

- 成果名称
- 类型
- 状态
- 驳回理由
- 所属项目
- 可见范围
- 创建时间

可执行操作：

| 操作 | 说明 |
| :--- | :--- |
| 查看详情 | 进入成果详情页 |
| 编辑 | 进入成果编辑页面 |
| 修改可见范围 | 当前页面按钮已保留，但功能提示为开发中 |

![图 4-7 个人成果物列表](user-manual-assets/screenshots/08-my-results.png)

### 4.4 编辑成果

操作路径：`个人成果物` -> `编辑`

编辑页面与创建页面类似，支持修改基础信息、动态字段、附件和可见范围。完成后点击`保存修改`。

注意事项：

- 对于被驳回成果，建议先在详情页或个人成果物列表查看驳回理由，再进入编辑页面修改。
- 编辑保存后，以系统实际状态流转为准。

### 4.5 浏览科研成果

操作路径：`科研成果`

用户可在科研成果页面浏览成果卡片，并按以下条件筛选：

- 成果类型
- 年份范围：全部、近1年、近3年、近5年

点击成果卡片可进入成果详情页。

![图 4-8 科研成果列表](user-manual-assets/screenshots/10-results-list.png)

### 4.6 成果检索

操作路径：`成果检索`

成果检索支持组合条件查询：

| 查询条件 | 说明 |
| :--- | :--- |
| 成果名称 | 输入成果标题或标题片段 |
| 关键词 | 输入一个或多个关键词，多个关键词可用空格分隔 |
| 成果类型 | 从已启用成果类型中选择 |
| 作者 | 输入作者姓名 |
| 年份范围 | 选择开始年份和结束年份 |
| 项目 | 输入项目名称或项目编号 |

操作步骤：

1. 输入或选择查询条件。
2. 点击`查询`。
3. 在结果列表中点击成果名称或`查看详情`。
4. 点击`重置`可清空条件并重新查询。

说明：

- 管理员在该页面可看到`导出`按钮，普通用户和专家用户不显示导出按钮。
- 管理员点击`导出`后，系统按当前检索条件下载 Excel 文件。

![图 4-9 成果检索页面](user-manual-assets/screenshots/10b-search-results.png)

### 4.7 查看成果详情与申请全文访问

操作路径：成果列表或检索结果 -> `查看详情`

成果详情页展示：

- 成果标题、状态、来源
- 作者、年份、可见性
- 摘要
- 正文或动态扩展字段
- 附件材料
- 基本信息
- 所属项目
- 关键词
- 驳回反馈

#### 4.7.1 已授权访问

如果用户拥有完整权限，页面显示`已授权访问`，可查看正文并下载附件。

![图 4-10 成果详情：已授权访问](user-manual-assets/screenshots/09-result-detail-authorized.png)

#### 4.7.2 访问受限

如果成果内容受限，页面会显示`访问受限`。此时用户可执行：

1. 点击`申请查看全文`。
2. 在弹窗中填写申请理由，最多 200 字。
3. 点击`提交申请`。
4. 提交后状态变为`审核中`，等待管理员审批。

审批结果：

- 管理员通过后，用户可查看正文及下载附件。
- 管理员拒绝后，页面显示`已拒绝`或拒绝原因，用户可补充理由后重新申请。

![图 4-11 成果详情：访问受限](user-manual-assets/screenshots/11b-result-detail-apply-entry.png)

![图 4-12 申请查看全文弹窗](user-manual-assets/screenshots/11c-access-apply-dialog.png)

![图 4-13 成果详情：权限审核中](user-manual-assets/screenshots/11-result-detail-restricted.png)

### 4.8 需求洞察

操作路径：`需求洞察`

需求洞察页面用于查看外部企业、政府平台、招投标网站等来源的需求线索，并查看系统匹配的候选成果。

#### 4.8.1 查询需求

页面支持以下筛选条件：

| 条件 | 说明 |
| :--- | :--- |
| 关键词 | 查询需求标题、摘要、行业等内容 |
| 行业 | 选择行业分类 |
| 地域 | 选择需求地域 |
| 状态 | 未匹配、已匹配、跟进中 |
| 来源类型 | 政府/园区平台、招投标网站、企业官网/年报、垂直行业论坛/新闻 |

操作步骤：

1. 输入筛选条件。
2. 点击`查询`。
3. 点击`重置`可清空筛选条件。
4. 点击右上角`刷新`可重新加载列表。

#### 4.8.2 查看需求详情与匹配成果

1. 在需求列表中点击需求行或`查看匹配`。
2. 系统从右侧打开`需求详情与匹配`抽屉。
3. 查看需求摘要、智能摘要、关键词、来源链接和匹配成果。
4. 如需重新计算匹配结果，点击`重新匹配`。
5. 在匹配结果中点击`查看成果`可进入对应成果详情页。

说明：

- 匹配度以百分比或分值形式展示，仅作为辅助参考。
- 是否跟进需求仍需业务人员人工判断。

![图 4-14 需求洞察页面](user-manual-assets/screenshots/18-demand-insights.png)

## 5. 专家用户操作说明

### 5.1 查看待审核成果

操作路径：`成果审核`

专家审核页面包含两个页签：

| 页签 | 说明 |
| :--- | :--- |
| 待审核 | 展示分配给当前专家且尚未审核的成果 |
| 已审核 | 展示当前专家已处理的成果 |

待审核列表展示：

- 成果名称
- 类型
- 所属项目
- 提交人
- 提交时间

![图 5-1 专家成果审核列表](user-manual-assets/screenshots/21-expert-reviews.png)

### 5.2 审核成果

操作步骤：

1. 在`待审核`页签中找到目标成果。
2. 点击`审核`。
3. 系统弹出`审核成果`对话框。
4. 查看成果标题、类型、年份、所属项目、提交人、作者、关键词、摘要等信息。
5. 如需查看完整页面，点击`查看完整详情`。
6. 如成果带附件，可在附件列表中点击`下载`。
7. 在`审核意见`中填写意见。
8. 在`审核结果`中选择`通过`或`驳回`。
9. 点击`提交`。

注意事项：

- 审核意见为必填项。
- 选择`驳回`时，应明确说明退回原因，便于提交人修改。
- 提交成功后，系统提示`审核成功`并刷新列表。

![图 5-2 成果审核弹窗](user-manual-assets/screenshots/22-expert-review-dialog.png)

### 5.3 查看已审核记录

操作路径：`成果审核` -> `已审核`

已审核列表展示：

- 成果名称
- 类型
- 所属项目
- 审核结果
- 审核时间

审核结果显示为`通过`或`驳回`。

## 6. 管理员操作说明

### 6.1 科研看台

操作路径：`科研看台`

科研看台用于查看全局成果统计和趋势，包括：

- 总成果数
- 论文数
- 专利数
- 本月新增
- 成果分布图
- 成果年度趋势图

可点击`查看全部`进入`成果管理`页面。

![图 6-1 管理员科研看台](user-manual-assets/screenshots/12-admin-dashboard.png)

### 6.2 成果管理

操作路径：`科研看台` -> `最新入库成果` -> `查看全部`

说明：`成果管理`页面当前有独立页面和按钮入口，但不在管理员左侧`成果物`菜单中直接展示。管理员也可由系统配置的管理入口直接进入该页面。

成果管理页面支持按以下条件筛选：

| 条件 | 可选值 |
| :--- | :--- |
| 关键词 | 输入成果相关关键词 |
| 状态 | 待审核、审核中、退回修改、已发布、已驳回 |
| 来源 | 手工上传、过程管理系统 |

列表展示：

- 成果名称
- 类型
- 项目阶段
- 来源
- 格式审查状态
- 审核状态
- 创建时间

可执行操作：

| 操作 | 适用场景 | 说明 |
| :--- | :--- | :--- |
| 查看 | 所有成果 | 打开成果详情页 |
| 分配审核 | 过程管理系统来源且状态为待审核 | 选择审核人并分配 |

### 6.5 成果类型配置

操作路径：`系统配置` -> `成果类型配置`

成果类型配置用于维护系统可选择的成果类型，并为不同类型配置扩展字段。

#### 6.5.1 新增成果类型

1. 点击`新增类型`。
2. 填写`类型名称`，例如论文、专利、软著。
3. 填写`类型代码`，例如 `paper`。建议使用小写英文和下划线。
4. 填写`描述`。
5. 设置`启用状态`。
6. 点击`立即创建`。

注意事项：

- 类型名称和类型代码为必填项。
- 类型代码是系统内部唯一编码，创建后编辑时不可修改。
- 如果类型代码重复或网络异常，系统会提示创建失败。

#### 6.5.2 编辑成果类型

3. 点击`保存修改`。

说明：

- 编辑时`类型代码`不可修改。
- 停用后的类型不会在成果创建页面展示。

#### 6.5.3 配置动态字段

1. 在成果类型列表中点击`配置字段`。
2. 系统打开动态字段配置窗口。
3. 点击`新增字段`添加一行字段。
4. 填写字段信息。
5. 点击`保存配置`。

字段配置项：

| 字段 | 说明 |
| :--- | :--- |
| 字段编码 | 系统内部字段编码，例如 `paper_title`。已保存字段的编码不可编辑 |
| 字段名称 | 页面展示名称，例如论文标题 |
| 字段类型 | 可选 `TEXT`、`NUMBER`、`BOOLEAN`、`DATE`、`MEDIA`、`JSON`、`EMAIL`、`RICHTEXT`、`DATETIME` |
| 说明 | 字段帮助说明 |

注意事项：

- 字段编码和字段名称不能为空。
- 当前配置页面保存字段时默认按必填字段处理。
- 删除已保存字段前，系统会弹出确认提示。
- 动态字段保存后，会影响对应成果类型的创建和详情展示。

## 8. 推荐业务流程

### 8.1 成果提交到审核通过

1. 普通用户进入`创建成果物`。
2. 选择成果类型。
3. 填写基础信息和动态字段。
4. 上传附件并设置可见范围。
5. 跳过当前未开放的智能补全步骤。
6. 确认信息并提交审核。
7. 管理员在`审核分配`或`成果管理`中分配审核人。
8. 专家在`成果审核`中查看成果并提交审核意见。
9. 如审核通过，成果进入已发布状态。
10. 如审核驳回，提交人在`个人成果物`中查看驳回理由并修改。

### 8.2 受限成果全文访问

1. 用户在成果详情页发现访问受限。
2. 用户点击`申请查看全文`并填写申请理由。
3. 管理员进入`权限审核`查看申请。
4. 管理员选择通过或拒绝。
5. 通过后，用户可查看正文和下载附件。
6. 拒绝后，用户可根据拒绝原因重新提交申请。

### 8.3 过程成果查看

1. 过程管理系统通过接口同步过程成果。
2. 管理员进入`过程成果`。
3. 按项目、类型或关键词筛选成果。
4. 查看过程成果详情。
5. 下载或预览附件。
6. 如存在来源链接，可跳转过程管理系统查看原始信息。

## 9. 常见问题

### 9.1 登录后看不到某个菜单怎么办？

请确认当前账号是否具备对应角色。不同角色可见菜单不同。例如普通用户不能访问管理员菜单，专家用户需要具备`research_expert`角色才能进入`成果审核`。

### 9.2 创建成果时无法进入下一步怎么办？

请检查：

- 是否已选择成果类型。
- 成果标题、作者、年份是否已填写。
- 当前成果类型的必填动态字段是否已填写。

### 9.3 附件上传失败怎么办？

请检查：

- 单个附件是否超过 20MB。
- 文件是否正在被本地软件占用。
- 网络连接是否正常。
- 如仍失败，请联系管理员或实施人员查看后端上传服务。

### 9.4 为什么看不到成果正文或附件？

该成果可能为受限成果。请在成果详情页点击`申请查看全文`，填写申请理由并等待管理员审批。

### 9.5 申请全文后为什么还是不能查看？

可能存在以下情况：

- 申请仍处于`审核中`。
- 管理员已拒绝申请。
- 页面数据未刷新，可点击详情页右上角刷新按钮或重新进入页面。

### 9.6 审核人列表为空怎么办？

管理员在分配审核人时，如果审核人下拉列表为空，请确认系统中是否已配置专家或管理员账号，并确认账号角色是否正确。

### 9.7 成果类型在创建页面不显示怎么办？

请管理员进入`成果类型配置`检查：

- 类型是否已创建。
- 类型是否处于启用状态。
- 类型是否被删除或停用。

### 9.8 动态字段保存失败怎么办？

请检查：

- 字段编码是否为空。
- 字段名称是否为空。
- 字段编码是否与已有字段冲突。
- 网络连接是否正常。



## 10. 当前版本功能边界

为避免上线培训和验收过程中产生误解，当前版本应按以下边界说明：

| 功能 | 当前状态 | 建议说明 |
| :--- | :--- | :--- |
